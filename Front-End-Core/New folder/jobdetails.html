Job Description

We are looking for a Senior Frontend Developer to join our team. The ideal candidate will have experience with React, TypeScript, and modern web technologies.

Responsibilities:

  • Develop and maintain web applications
  • Collaborate with UX designers to implement designs
  • Optimize application performance

Requirements:

  • Proficiency in React and TypeScript
  • Strong understanding of HTML, CSS, and JavaScript
  • Experience with modern build tools and version control
  • Excellent problem-solving and communication skills
